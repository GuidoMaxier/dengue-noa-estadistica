# Análisis Epidemiológico de Dengue en el Noroeste Argentino (NOA)  
### Período: 2018-2025

---

## Descripción del Proyecto
Este repositorio contiene un análisis estadístico completo de los casos de dengue registrados en las provincias del Norte Argentino (NOA) entre 2018 y 2025.  
El estudio integra estadística descriptiva e inferencial para detectar tendencias epidemiológicas, estacionalidad, variaciones geográficas e impacto contextual.

---

## Objetivos

- Analizar el comportamiento epidemiológico del dengue en NOA  
- Identificar patrones temporales y distribución geográfica  
- Evaluar el impacto de la pandemia COVID-19 en la incidencia  
- Desarrollar modelos predictivos para años futuros
